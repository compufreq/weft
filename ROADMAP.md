# Weft Roadmap

Release management for Weft. Versioning follows [SemVer](https://semver.org); every phase closes with a full test gate (unit + integration + security + UI/UX) and an annotated git tag that triggers the release pipeline.

## Release & versioning policy

- **Pre-1.0:** each phase lands as a minor (`v0.x.0`); fixes as patches (`v0.x.y`)
- **Tags:** annotated `vX.Y.Z` on `main` ‚Üí CI builds multi-arch images ‚Üí GHCR (`X.Y.Z`, `X.Y`, `latest`) ‚Üí GitHub Release with git-cliff notes
- **Commits:** [Conventional Commits](https://www.conventionalcommits.org) enforced; `CHANGELOG.md` generated by git-cliff
- **Weaviate support policy:** latest two Weaviate minor lines tested in CI; best-effort back to 1.30

## Phases

### v0.1.0 ‚Äî Walking skeleton *(shipped 2026-07-16)*
Monorepo, dockerized dev stack with seeded Weaviate, `weft-weaviate` client (meta + schema), axum API (`/healthz`, instances, schema), SSR schema browser page, CI + security pipelines, quickstart docs.

### v0.2.0 ‚Äî Schema & connections *(shipped 2026-07-16)*
Multi-instance registry (config + runtime add), schema detail views (properties, vectorizer, index config), schema export + diff (instance‚Üîinstance, instance‚Üîfile), **all-in-one zero-config image on GHCR**, docs site on GitHub Pages.

### v0.3.0 ‚Äî Data explorer *(shipped 2026-07-16)*
Cursor-paginated object browser (virtualized), JSON detail drawer, BM25 / nearVector / nearText / hybrid search with scores, NDJSON streaming export.

### v0.4.0 ‚Äî Multi-tenancy *(shipped 2026-07-16)*
Tenant list/filter, HOT/COLD activation, per-tenant object counts, tenant-scoped explorer, bulk tenant operations.

### v0.5.0 ‚Äî Ops dashboard *(shipped 2026-07-16)*
Node/shard health (live polling), index & queue status, backup create/restore (module-gated), Weaviate version & capability display.

### v0.6.0 ‚Äî Hardening *(shipped 2026-07-16)*
Optional API auth (`WEFT_AUTH_TOKEN`), read-only mode, Weaviate version test matrix (1.30 + latest) with graceful degradation, 100k-object performance validation.

### v0.7.0 ‚Äî Auth hardening *(shipped 2026-07-16)*
Session rate limiting (5 attempts/min/IP on `POST /api/v1/auth/session`, 429 + `Retry-After`), logout endpoint (`DELETE /api/v1/auth/session`) with a nav **Log out** button when auth is active.

### v0.8.0 ‚Äî Multi-arch & docs *(shipped 2026-07-16)*
arm64 image (native GitHub arm runners, multi-arch manifest on GHCR), docs review (feature lists refreshed) + upgrade guide, docs-site a11y fixes (code wrap, focusable scroll regions).

### v0.9.0 ‚Äî Explorer completeness *(shipped 2026-07-16)*
Where-filter builder on the object browser and search, aggregations (counts + per-property facets), raw GraphQL query console (read-safe: Weaviate's GraphQL API is query-only), read-only-mode fix for non-mutating POST endpoints.

### v0.10.0 ‚Äî Data write path *(shipped 2026-07-16)*
Object create / edit / delete from the UI, JSON/NDJSON batch import with per-item error reports ‚Äî all blocked in read-only deployments (CSV import shipped in v1.1.0).

### v0.11.0 ‚Äî Schema management & aliases *(shipped 2026-07-16)*
Collection create (guided form + raw JSON), add property, collection delete behind typed confirmation, collection alias CRUD (Weaviate ‚â•1.32, graceful degradation below), persisted runtime instances (`WEFT_INSTANCES_FILE`).

### v0.12.0 ‚Äî Ops depth *(shipped 2026-07-17)*
RBAC viewer (roles, users, assignments ‚Äî read-only, graceful when RBAC is off), Raft cluster-statistics panel, vector-space map (2D PCA projection with facet coloring, no external libs).

### v1.0.0 ‚Äî Stable *(shipped 2026-07-17)*
Security review (threat-model page over the full mutation surface), `/api/v1` stability promise, complete documentation.

CSV import (client-side RFC-4180 parsing, schema-typed columns, `id`/`vector` special columns, file picker), AND/OR + nested groups in the filter builder (additive `/api/v1` extension ‚Äî the flat filter shape is unchanged), dark/light theme toggle with OS-preference fallback.
